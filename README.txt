1. Файл greetings.py
Программа выводит в консоль данные из переменных 
"Имя пользователя"
"Название заметки"
"Описание заметки"
"Статус заметки"
"Дата создания заметки"
"Дата истечения заметки"

2. Файл add_input.py
Программа запрашивает у пользователя данные, которые вводятся через консоль, и записывает их в переменные.

3. Файл date_changer.py
Программа преобразует дату из фотмата "ДД.ММ.ГГГГ" в формат "ДД.ММ". При этом в переменных остается полная дата.

4. Файл add_list.py
Переменная titles записана в виде списка. Пользователь вводит название заметки и два подзаголовка, которые потом выводятся в консоль.

5. Файл final.py
Переменные с информацией о заметке записаны в один список. Программа выводит в консоль список, состоящий из всех переменных с информацией о заметке.
